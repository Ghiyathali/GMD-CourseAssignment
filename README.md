# Lab Escape

An infinite runner game built in Unity 6 for a Game Development course. You play as a human test subject escaping from a research laboratory, being chased by a security robot. Run through an endless procedurally generated lab corridor, dodge obstacles, collect blood bags, and survive as long as possible.

## Controls

| Action | Keyboard | Arcade / Gamepad |
|---|---|---|
| Move Left | A / Left Arrow | Left Stick Left |
| Move Right | D / Right Arrow | Left Stick Right |
| Jump | Space / W / Up Arrow | Button North (Y) |
| Slide | S / Down Arrow | Button South (A) |
| Pause | Escape | Start Button |
